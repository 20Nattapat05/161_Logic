หน่วยที่: 4 การลดรูปสมการบูลีน

1. f(A,B) = Σm(0,1,2)
m0=00, m1=01, m2=10 → ทุกกรณียกเว้น 11
จะเป็น 0 แค่ A=1,B=1 ⇒ f = A̅ + B̅

คำตอบ: f = A̅ + B̅

2. f(A,B) = ΠM(0,1,2)
เป็น 0 ที่ 00,01,10 และเป็น 1 แค่ 11 ⇒ f = AB

คำตอบ: f = AB

3.1 f(A,B,C) = Σm(0,1,2,3,4,5)
เป็น 0 แค่ 110 และ 111 ⇒ A=1,B=1 ⇒ f = A̅ + B̅

คำตอบ: f = A̅ + B̅

3.2 f(A,B,C,D) = ΠM(0,1,2,3,7,8,9,10,11,15)
ตัวที่เป็น 1 คือ 4,5,6,12,13,14
จัดกลุ่ม K-map ⇒ f = BC̅ + ACD̅

คำตอบ: f = BC̅ + ACD̅

4.1 f(A,B,C,D) = Σm(1,3,5,7,9)
ทุกตัวมี D = 1
จัดกลุ่ม K-map จะได้กลุ่ม 1,3,5,7 ⇒ A̅D และกลุ่ม 1,9 ⇒ B̅C̅D
ดังนั้น f = A̅D + B̅C̅D = D(A̅ + B̅C̅)

คำตอบ: f = D(A̅ + B̅C̅)

4.2 f(A,B,C,D) = ΠM(0,2,4,6,8)
ทำ K-map จะได้กลุ่มของค่า 1 ครอบคลุมทุกช่องที่ B=1, C=1 หรือ D=1
จึงสามารถลดรูปได้เป็น f = B + C + D

คำตอบ: f = B + C + D
